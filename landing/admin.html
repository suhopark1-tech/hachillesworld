Google Analytics 4 설정 가이드

GA4를 연동하면 지역별·시간대별·디바이스별 방문자, 블로그 체류시간, 유입 채널을 Google Analytics 대시보드에서 실시간으로 확인할 수 있습니다. 무료입니다.

1
GA4 속성 생성
analytics.google.com → 관리 → 속성 만들기 → 웹 스트림 추가 → suhopark1-tech.github.io/hachillesworld 입력
2
Measurement ID 복사
웹 스트림 세부정보에서 G-XXXXXXXXXX 형식의 ID를 복사합니다.
3
haw-analytics.js 수정
landing/haw-analytics.js 파일 최상단의
GA4_ID: 'G-E78PZ2H8FG' — 설정 완료.
4
커밋 & 배포
git add landing/haw-analytics.js && git commit -m "feat: GA4 연동" && git push
GitHub Pages CI가 자동 배포합니다. (~30초)
5
확인
GA4 대시보드 → 실시간 리포트에서 방문자가 표시되면 완료입니다.
지역·시간 분석은 데이터 누적 후 (보통 24~48시간) 확인 가능합니다.
Google Analytics 열기 → haw-analytics.js 보기
고급: Docker 스택 (Prometheus · Grafana · ClickHouse)

로컬에서 docker compose up을 실행하면 FastAPI(8000) · Prometheus(9090) · Grafana(3000)가 기동됩니다. Grafana에서 진단 API 메트릭과 시스템 상태를 실시간으로 시각화할 수 있습니다.
