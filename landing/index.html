Framework

Levels × Laws
진단 좌표계

Product

세 가지 모듈, 하나의 플랫폼

진단부터 최적화, 실시간 운영까지 에이전트 생애주기 전체를 커버합니다.

MODULE 01

HAchillesWorld Scan

진단 엔진

에이전트 시스템을 연결하면 10분 안에 15개 지표 기반 진단 리포트를 생성합니다. 현재 Level과 Laws 도메인 위치, 핵심 병목을 즉시 파악하세요.

  • World Model 품질 5개 지표 (ECE, Drift Rate 등)
  • 에이전시 수준 5개 지표 (Planning Depth, Self-correction)
  • 운영 건전성 5개 지표 (비용 효율, HITL 빈도)
  • SDK / OTel 브리지 / 로그 업로드 3가지 연동
MODULE 02

HAchillesWorld Optimize

최적화 엔진

Scan 결과를 기반으로 맞춤형 최적화 로드맵을 자동 생성합니다. 4주·12주·24주 단계별 실행 계획과 하네스 규칙 코드를 즉시 받아보세요.

  • 영향도 × 난이도 우선순위 매트릭스 자동 생성
  • Laws 도메인 맞춤 하네스 규칙 코드 자동 생성
  • L1→L2, L2→L3 전환 표준 로드맵
  • 예상 ROI 계산서 (비용 절감, 실패 감소)
MODULE 03

HAchillesWorld Operate

운영 인텔리전스

프로덕션 에이전트의 World Model 건전성을 실시간 모니터링합니다. 드리프트 경보, Replay Debugging, Meta-Harness 자동화로 사전에 위험을 차단하세요.

  • 실시간 Simulation Drift 감지 + 자동 재보정
  • 실패 에피소드 단계별 Replay Debugging
  • Meta-Harness 자동 규칙 생성 + 검토 워크플로
  • Slack / Teams 알림 통합
