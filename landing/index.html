Why HAchillesWorld

에이전트를 배포했지만
왜 실패하는지 모른다면

기존 MLOps·APM 도구는 모델 성능과 인프라를 측정합니다. 에이전트가 세계를 얼마나 이해하는지는 아무도 측정하지 않습니다.

🗺️

"L3를 목표로 하는데 어디서 시작해야 하는가"

💸

"AI 비용이 폭발하는데 어디서 새는지 모른다"

라우팅 최적화 부재, 불필요한 대형 모델 과다 사용. AI ROI가 악화됩니다.

🔍

Scan

10분 진단

Levels × Laws 좌표계에서 에이전트의 현재 위치를 자동 판정합니다. 15개 지표로 World Model 품질, 에이전시 수준, 운영 건전성을 측정합니다.

  • L1 / L2 / L3 자동 Level 판정
  • Physical·Digital·Social·Scientific Laws 도메인 분류
  • Simulation Drift Rate, ECE 등 15개 지표 계산
  • 종합 점수 0~100 + 즉시 조치 항목 목록
  • CLI·SDK·로그 파일 3가지 입력 방식
🗺️

Optimize

자동 로드맵 생성

진단 결과를 기반으로 맞춤형 최적화 로드맵을 자동 생성합니다. 하네스 규칙을 자동으로 설계하고 Python 코드로 출력합니다.

  • L1→L2·L2→L3 단계별 실행 로드맵
  • Phase별 우선 태스크·기간·예상 점수 향상
  • 하네스 규칙 자동 생성 (Python 코드 출력)
  • 비용 절감 기회 분석 및 CostAwareRouter 설정
  • ROI 추정 (연간 절감액)
⚡

Operate

실시간 운영 지원

프로덕션 에이전트의 World Model 건전성을 실시간으로 감시합니다. 실패 에피소드를 단계별로 재생해 근본 원인을 10분 안에 찾습니다.

  • DriftMonitor — 실시간 Simulation Drift 감지
  • 경보 콜백으로 Slack·Teams·PagerDuty 연동
  • Replay Debugger — 실패 에피소드 단계별 재생
  • Meta-Harness — 실패 패턴 자동 학습 및 규칙 제안
  • 이상 프레임 자동 하이라이트 + 근본 원인 판정

How It Works

5분 안에
첫 진단 결과

에이전트 클래스에 데코레이터 하나, 또는 기존 로그 파일을 업로드하면 즉시 시작됩니다.

1

SDK 설치 (30초)

pip으로 설치 후 API 키 없이도 로컬에서 전체 기능 사용 가능.

pip install hachillesworld
2

에이전트에 계측 추가 (2분)

@instrument 데코레이터 하나로 plan·execute·observe·reflect 자동 기록.

@instrument(client, agent_name="my-agent")
class MyAgent:
    def plan(self, state, goal): ...
3

Scan 실행 (10분)

로그 수집 후 15개 지표 자동 계산, Level 판정, 리포트 생성.

hachillesworld scan \
  --logs ./logs.json \
  --name "my-agent"
4

로드맵 수신 및 실행

진단 결과 기반 맞춤 로드맵과 하네스 코드를 자동 생성합니다.

hachillesworld optimize ./report.json

# 에이전트에 HAchillesWorld 계측 추가 (3줄) from hachillesworld import HAchillesWorldClient, instrument client = HAchillesWorldClient() # API 키 없이 로컬 실행 가능 @instrument(client, agent_name="my-agent") class MyAgent: def plan(self, state, goal): # 기존 코드 그대로 — HAchillesWorld가 자동 계측 return self._select_action(state, goal) def execute(self, action): return self._run(action) # 진단 실행 report = client.scan(logs=logs, config=config, agent_name="my-agent") print(report.summary()) # 🟡 [my-agent] L2.3 × Digital Laws — 종합 74/100
# 진단 + 로드맵 + 하네스 자동 생성 from hachillesworld import HAchillesWorldClient from hachillesworld.optimize.roadmap import RoadmapGenerator from hachillesworld.optimize.harness_generator import HarnessGenerator client = HAchillesWorldClient() report = client.scan(logs=logs, config=config, agent_name="my-agent") # 1. 진단 결과 print(f"Level: {report.level_label}") # L2.3 print(f"Score: {report.composite_score}/100") # 74/100 # 2. 최적화 로드맵 자동 생성 roadmap = RoadmapGenerator().generate(report, target_level="L3") print(f"총 기간: {roadmap.estimated_duration_weeks}주") # 24주 # 3. 하네스 규칙 자동 생성 spec = HarnessGenerator().generate(report) print(spec.to_python()) # 즉시 사용 가능한 Python 클래스 출력
# 실시간 Simulation Drift 감지 from hachillesworld.operate.monitor import DriftMonitor monitor = DriftMonitor( agent_name="production-agent", threshold=0.15, # 이 값 초과 시 드리프트 alert_rate_threshold=0.20 # 윈도우의 20% 이상이면 경보 ) # 경보 핸들러 — Slack·PagerDuty 등 연결 가능 monitor.on_alert = lambda alert: send_slack( f"⚡ Drift {alert.drift_value:.3f} — {alert.recommended_action}" ) # DEOR 루프에서 매 스텝 호출 for step in range(n_steps): predicted = world_model.predict(state, action) actual = env.observe() drift = monitor.record(predicted, actual) if drift > monitor.threshold: world_model.recalibrate(actual) # 자동 재보정
