mcts_planner.py — HCE M3 핵심 실습 (4단계 완전 구현) ⭐ M3 핵심
from hachillesworld.optimize.mcts import MCTSPlanner, MCTSNode

# HCE 목표: Depth 15 · n_simulations 50 · latency ≤ 200ms
planner = MCTSPlanner(
  agent_name="hce-cert-agent",
  n_simulations=50,
  exploration_c=1.414,   # UCB1 √2
  max_depth=15,         # HCE 달성 기준
  rollout_policy="random",
  discount_factor=0.95,
)

# 4단계: Select → Expand → Rollout → Backprop
root = MCTSNode(state=initial_state)
for _ in range(planner.n_simulations):
  node  = planner.select(root)               # UCB1 탐색
  child = planner.expand(node, world_model)   # 자식 확장
  rew   = planner.rollout(child, world_model)  # 시뮬레이션
  planner.backpropagate(child, rew)           # 보상 전파
best_action = planner.best_action(root)

# HCAN 업그레이드: Neural Rollout + Depth 22
# MCTSPlanner(max_depth=22, n_simulations=200,
#  rollout_policy=NeuralRolloutPolicy(world_model))

FAQ

자주 묻는 질문

HCP 없이 HCE에 바로 도전할 수 있나요? ▼
원칙적으로 순서를 권장합니다. 실무 경험 2년 이상인 경우 HCP 면제 신청이 가능합니다(심사 비용 ₩50,000). 단, HCP 내용은 HCE 시험에 포함되므로 충분한 이해가 필요합니다.
시험은 온라인으로 응시할 수 있나요? ▼
이론 시험과 실습 제출은 모두 온라인으로 진행합니다. HCAN 구술 심사만 화상 미팅으로 진행되며, 월 2회 일정이 제공됩니다.
개인 업무 데이터를 실습에 사용해도 되나요? ▼
가능합니다. 제출 전 개인정보·영업비밀을 반드시 익명화해야 합니다. 제공되는 샘플 데이터셋으로도 모든 실습을 완료할 수 있습니다.
HCAN 갱신(1년)을 놓치면 어떻게 되나요? ▼
만료 후 90일 이내에는 갱신 절차로 복원 가능합니다. 90일 초과 시 HCE부터 재취득이 필요합니다. 갱신 조건은 시험(40문항) + 사례 제출 + 커뮤니티 기여 중 1가지입니다.
에이전트 L3 인증과 개인 HCAN 인증은 다른가요? ▼
다릅니다. 개인 HCAN은 사람의 역량 인증이고, 에이전트 L3 인증은 시스템의 품질 인증입니다. HCAN 보유자는 에이전트 L3 인증 심사에 검토위원으로 참여할 수 있습니다.
학생·오픈소스 기여자 할인이 있나요? ▼
학생·취준생은 학생증 제출 시 50% 할인, GitHub hachillesworld 저장소 Merged PR 1건 이상 기여자는 30% 할인 쿠폰을 제공합니다. 팀 5인 이상은 20% 단체 할인이 적용됩니다.
