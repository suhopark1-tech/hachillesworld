Phase 3 Week 21 생성 · 승인 후 적용

Counterfactual 시뮬레이션 결과 후회값(regret)이 0.5 이상인 결정은 즉시 중단하고 상위 대안으로 전환합니다.

def check_cf_regret(self, cf_results, proposed_action): regret = cf_results.compute_regret(proposed_action) if regret > 0.50: best = cf_results.best_alternative() return Rule.REDIRECT( reason=f"후회값 {regret:.2f} > 0.50 — 대안 {best.name}으로 전환", alternative_action=best ) return Rule.PASS()
