Planning Depth와
비용의 상관관계

더 멀리 내다보는 에이전트가 왜 더 저렴한가 — 수치로 증명

박
박성훈
2026년 6월 약 3,000자 · 11분 읽기 ◈ 데이터 분석

직관과 반대되는 현상이 있다. Planning Depth를 높이면 — 즉 에이전트가 더 많은 미래 시나리오를 시뮬레이션하게 만들면 — 계산량이 늘어나므로 비용이 올라갈 것 같다.

실제로는 정반대다.

HAchillesWorld로 12개 에이전트의 6개월 운영 데이터를 분석한 결과, Planning Depth와 LLM 호출 비용 사이에는 강한 음의 상관관계(r = −0.87)가 존재했다. Depth가 높아질수록 비용이 내려간다.

왜 이런 역설이 발생하는가

Planning Depth가 낮은 에이전트는 근시안적이다. 3스텝 앞밖에 못 보기 때문에, 4스텝 후에 문제가 생기면 그 시점에서 다시 LLM에게 물어야 한다. 문제를 만날 때마다 재호출이 발생한다.

Planning Depth가 높은 에이전트는 처음 계획을 세울 때 15스텝 후까지 시뮬레이션한다. 중간에 발생할 문제를 미리 파악하고 대안을 계획에 포함시킨다. 실행 중에 재호출할 일이 줄어든다.

● Depth별 호출 패턴 비교 
Depth 3 에이전트 (태스크 1건):
  계획 수립      → LLM 호출 1회
  문제 발생 #1   → 재호출 2회
  문제 발생 #2   → 재호출 3회
  결과 검증      → 재호출 1회
  합계: 7회

Depth 15 에이전트 (동일 태스크):
  계획 수립      → LLM 호출 1회 (문제 #1, #2 선제 처리 포함)
  결과 검증      → 재호출 1회
  합계: 2회

같은 태스크를 처리하는 데 Depth 3은 7회, Depth 15는 2회 호출한다. 비용 차이는 3.5배다.

실측 데이터: r = −0.87

HAchillesWorld가 수집한 12개 에이전트, 6개월 데이터를 분석했다. 에이전트마다 Planning Depth와 태스크당 평균 LLM 호출 수를 측정했다.

Depth 3 에이전트(100%)와 Depth 22 에이전트(22%)의 비용 차이는 4.5배다. 피어슨 상관계수 r = −0.87은 통계적으로 유의미한 강한 음의 상관관계를 나타낸다.

임계점: Depth 8~12 구간

그래프를 보면 흥미로운 패턴이 있다. Depth가 낮을 때는 1포인트 증가당 비용 절감 효과가 크다. 그러나 Depth 8~12 구간을 넘어서면 한계 효율이 줄어든다.

● Depth 구간별 한계 절감율 
Depth  3→4  : −7%   (1포인트당)
Depth  4→6  : −11.5%
Depth  6→8  : −8.5%
Depth  8→11 : −11%  ← 효율 최고 구간
Depth 11→14 : −4.7%
Depth 14→18 : −3.1%
Depth 18→22 : −1.9%  ← 한계 체감

대부분의 엔터프라이즈 에이전트에서 Depth 10~15 구간이 비용 대비 효과의 스위트 스팟이다. 이 구간을 목표로 최적화하면 비용을 60~70% 줄이면서 추가 투자 대비 수익이 가장 높다.

주의: Depth를 높이는 것도 비용이 든다

Planning Depth를 높이는 과정 자체도 비용이 든다. MCTS 기반 시뮬레이션이 더 많은 계산을 요구하기 때문이다. 따라서 단순히 Depth를 최대화하는 것이 정답이 아니다.

HAchillesWorld Optimize가 계산하는 것이 바로 이 트레이드오프다. 현재 Depth에서 다음 단계로 올리는 데 드는 비용과, 그로 인해 절감되는 LLM 호출 비용을 비교해서 순편익이 양수인 구간까지만 최적화를 권장한다.

● HAchillesWorld Optimize — 비용 분석 예시 
현재 상태: Depth 4, 월 LLM 비용 ₩930만

권장 로드맵:
  → Depth  4→8  : 구현 비용 ₩80만 / 월 절감 ₩340만  ✅ ROI 4.3x
  → Depth  8→12 : 구현 비용 ₩60만 / 월 절감 ₩150만  ✅ ROI 2.5x
  → Depth 12→18 : 구현 비용 ₩90만 / 월 절감 ₩90만   ⚠ ROI 1.0x (보류)
  → Depth 18→22 : 구현 비용 ₩70만 / 월 절감 ₩40만   ❌ ROI 0.6x (비권장)
