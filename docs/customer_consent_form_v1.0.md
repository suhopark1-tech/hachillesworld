## 동의서 전문

**HAchillesWorld 운영사** (이하 "회사")는 AI 에이전트 World Model 진단 플랫폼(이하 "서비스")을 제공하기 위하여 귀하의 개인정보를 수집·이용합니다.

회사는 「개인정보보호법」 제15조 및 제22조, EU 일반개인정보보호규정(GDPR) 제6조에 따라 귀하의 동의를 받아 개인정보를 처리합니다. 아래 내용을 충분히 읽으신 후 동의 여부를 결정해 주십시오.

## Part 2. 선택 동의 A

### 익명 벤치마크 집계 기여

**이 동의는 선택사항입니다. 동의하지 않으셔도 서비스를 정상적으로 이용하실 수 있습니다.**

---

### 2-1. 이 동의가 필요한 이유

HAchillesWorld의 핵심 가치 중 하나는 **"내 에이전트가 업계에서 어느 수준인가?"** 를 알려드리는 것입니다.

이 비교를 위해서는 다양한 고객의 진단 데이터를 **완전히 익명화하여** 집계한 벤치마크 데이터셋이 필요합니다.  
귀하가 이 동의를 해주시면, 귀하의 진단 결과가 업계 평균 산출에 기여되며 동시에 귀하도 더 정확한 업계 비교 데이터를 받으실 수 있습니다.

---

### 2-2. 수집·활용 항목

| 활용 항목 | 익명화 처리 방법 |
|---------|--------------|
| HAS 점수 (0~100) | 통계적 노이즈 처리 후 집계 |
| 15개 진단 지표 수치값 | 수치만 추출, 원시 데이터 제거 |
| 에이전트 운영 도메인 | 금융·의료 등 카테고리만 |
| 에이전트 레벨 (L1/L2/L3) | 레벨 분류값만 |

---

### 2-3. 익명화 보장 내용

회사는 다음 기술적 조치를 통해 귀하의 신원이 절대 식별되지 않음을 보장합니다.

```
✅ 식별 정보 완전 제거
   고객 ID, 에이전트 ID, 회사명 → 모두 삭제 후 집계

✅ 차분 프라이버시 (Differential Privacy) 적용
   수치값에 통계적 노이즈를 추가하여 역추적 불가 처리

✅ K-익명성 보장
   동일 도메인·레벨 조합에서 최소 5개 이상 에이전트 데이터가
   모이지 않으면 집계를 생략합니다 (단독 식별 방지)

✅ 원시 데이터 미보관
   집계 후 개별 기여 데이터는 보관하지 않습니다
```

---

### 2-4. 활용 범위 및 제공 형태

| 활용 목적 | 구체적 형태 | 공개 여부 |
|---------|-----------|:-------:|
| 플랫폼 내 비교 | "귀사 에이전트: 금융 도메인 상위 35%" | 고객 본인만 |
| 공개 벤치마크 API | 도메인별 HAS 분포 통계 (집계값만) | 공개 |
| 웹사이트 벤치마크 페이지 | 업종별 평균·중앙값 시각화 | 공개 |
| 내부 제품 개선 | 기능 우선순위 결정 | 비공개 |

> **개별 고객 데이터는 어떤 형태로도 공개되지 않습니다.**

---

### 2-5. 동의 철회 시 처리

동의를 철회하시면 이후 진단 결과는 집계에서 제외됩니다.  
단, **이미 집계된 통계값**은 특정 고객을 식별하는 것이 기술적으로 불가능하므로 소급하여 제거할 수 없습니다. 이 점 양해 부탁드립니다.

---

### [선택 A] 동의 확인

```
□  저는 위 내용(2-1 ~ 2-5)을 충분히 읽고 이해하였으며,
   제 에이전트 진단 데이터를 완전 익명화하여 업계 벤치마크
   집계에 기여하는 것에 동의합니다.

   동의 시 혜택:
   · 도메인 내 상위 몇 % 인지 즉시 확인 가능
   · 업계 평균 대비 취약 지표 자동 하이라이트
   · 더 정확한 개선 우선순위 추천
```

## Part 3. 선택 동의 B

---

### 3-1. 수집 항목 및 목적

서비스를 더 편리하고 직관적으로 개선하기 위해 플랫폼 내 행동 데이터를 수집합니다.

| 수집 항목 | 목적 |
|---------|------|
| 페이지 방문 기록 | 자주 사용하는 기능 파악 |
| 스캔 시작·완료 시각 | 진단 소요 시간 최적화 |

| 이탈 지점 | 불편한 단계 개선 |

---

### 3-2. 수집하지 않는 항목

```
수집하지 않는 항목:
  ✗ 에이전트의 실제 처리 내용 (텍스트, 문서 등)
  ✗ 키보드 입력값 또는 화면 내 텍스트
  ✗ 로그인 정보 또는 인증 데이터
  ✗ 타 웹사이트 방문 기록
```

---

### 3-3. 보관 및 처리

- 수집된 행동 데이터는 **수집 후 1년** 보관 후 자동 파기

- 외부 마케팅 플랫폼에 제공하지 않습니다

## Part 5. 선택 동의 D

---

### 5-1. 이 동의의 의미

귀하의 에이전트 개선 사례를 HAchillesWorld의 공개 케이스스터디로 게재하는 것에 대한 동의입니다.  
HAS 점수가 유의미하게 향상된 고객에게 별도 인터뷰 후 개별 제안합니다.

---

### 5-2. 활용 내용 및 범위

| 항목 | 공개 수준 선택 |
|------|-------------|
| 회사명 및 로고 | □ 공개 허용  □ 익명(업종만 공개) 희망 |
| HAS 점수 Before/After | □ 공개 허용  □ 비율(%)만 공개 희망 |
| 개선에 적용한 방법론 | □ 공개 허용  □ 요약본만 공개 희망 |
| 담당자 인터뷰 인용 | □ 공개 허용  □ 익명 처리 희망 |

---

### 5-3. 게재 위치 및 활용 범위

```
게재 위치:
  · HAchillesWorld 공식 웹사이트 /case-studies 페이지
  · 회사 블로그, LinkedIn 공식 채널
  · 영업·마케팅 자료 (귀하 사전 확인 후 사용)

활용 제한:
  · 게재 전 최종본 확인 및 수정 요청 권리 보장
  · 게재 후 철회 요청 시 14일 내 삭제 처리
  · 제3자 미디어 재판매 금지
```
