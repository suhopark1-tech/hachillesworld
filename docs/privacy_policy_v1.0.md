## 1. 개인정보의 처리 목적

회사는 다음의 목적으로 개인정보를 처리합니다. 처리하는 개인정보는 다음 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 「개인정보보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행합니다.

### 1-1. 서비스 제공 및 계약 이행 (법적 근거: 계약 체결·이행)

| 세부 목적 | 처리 내용 |
|---------|---------|
| 회원 가입 및 계정 관리 | 가입 신청 확인, 본인 식별, 불량 회원 제재, 계정 유지 및 탈퇴 처리 |
| AI 에이전트 진단 서비스 제공 | 에피소드 데이터 수신, HAS 점수 산출, 15개 지표 진단, DiagnosticReport 생성 |
| 플랜별 기능 제공 | 구독 플랜(Free·Starter·Pro·Enterprise) 기능 차등 제공 |
| 고객지원 | 이용 문의 처리, 불만 처리, 서비스 장애 대응 |

### 1-2. 보안 및 법적 의무 이행 (법적 근거: 법령상 의무)

| 세부 목적 | 처리 내용 |
|---------|---------|
| 보안 감사 | 부정 접근 탐지, API 호출 이상 감지, 감사 로그(AuditEvent) 유지 |
| 법령 의무 이행 | 수사기관의 적법한 요청 대응, 법원 명령에 따른 자료 제출 |
| 분쟁 해결 | 이용약관 위반 처리, 법적 청구권 행사·방어 |

### 1-3. 서비스 개선 (법적 근거: 동의)

이용자의 별도 동의를 받은 경우에 한하여 다음 목적으로 처리합니다.

| 동의 항목 | 세부 목적 |
|---------|---------|
| **제품 개선 분석 동의** | 서비스 이용 패턴 분석(퍼널 분석, 이탈 지점 파악), 기능 개선 |
| **익명 벤치마크 기여 동의** | 도메인별 AI 에이전트 성능 벤치마크 통계 산출 (완전 익명화 처리) |
| **마케팅 정보 수신 동의** | HAchilles Weekly 뉴스레터, 웨비나·이벤트 안내, 서비스 업데이트 발송 |
| **공개 케이스스터디 동의** | 성과 개선 사례 웹사이트·마케팅 자료 활용 (별도 인터뷰 후 추가 동의 필수) |

## 2. 처리하는 개인정보 항목

### 2-1. 회원 가입 시 수집 항목

| 항목 | 필수/선택 | 처리 방법 |
|------|:-------:|---------|
| 이메일 주소 | 필수 | 직접 입력, AES-256 암호화 저장 |
| 비밀번호 | 필수 | 직접 입력, bcrypt 해시 저장 (원본 불보관) |
| 회사명 (또는 팀명) | 선택 | 직접 입력, AES-256 암호화 저장 |
| 서비스 이용 지역 | 필수 | 자동 감지 또는 선택 (KR·US·EU·JP) |

### 2-2. 서비스 이용 중 자동 수집 항목

| 항목 | 수집 내용 | 처리 방법 |
|------|---------|---------|
| API 키 | 서비스 인증 수단 | bcrypt 해시 저장, 앞 8자리만 로그 표시 (원본 불보관) |
| 에이전트 식별 정보 | 에이전트 이름, 운영 도메인 (금융·의료·고객서비스 등) | 직접 입력 |
| 에이전트 진단 수치 데이터 | 에피소드 실행 결과 수치값 15개 지표 (HAS 점수, WMQ·ALM·OHM 카테고리) | SDK 자동 전송, PostgreSQL 저장 |
| 서비스 접속 기록 | 접속 일시, 서비스 이용 이력, API 호출 내역 | 자동 수집, AuditEvent 기록 |
| 접속 IP 주소 | 보안 감사 목적 | SHA-256 해시 처리 (원본 불보관), 감사 로그 마스킹 처리 |
| 브라우저 정보 (User-Agent) | 기기 유형 파악 | 자동 수집, **집계 후 30일 내 원본 삭제** |

### 2-3. 동의 시 추가 수집 항목

| 동의 항목 | 추가 수집 내용 |
|---------|-------------|
| 제품 개선 분석 동의 | 서비스 이용 행동 데이터 (페이지 조회, 스캔 시작·완료, 리포트 열람, 업그레이드 클릭 등) |
| 익명 벤치마크 기여 동의 | 진단 수치 데이터를 완전 익명화하여 도메인별 통계 집계에 활용 |
| 마케팅 정보 수신 동의 | 이메일 주소 (뉴스레터·이벤트 발송 목적) |
| 공개 케이스스터디 동의 | 인터뷰 내용, 에이전트 개선 성과 (별도 서면 동의 후) |

> **중요 안내**
>
> 회사는 귀하의 에이전트가 처리하는 **최종 사용자의 개인정보, 원문 텍스트, 소스 코드, 업무 데이터를 수집하지 않습니다.**  
> 에피소드 데이터 중 예측 상태(predicted_next_state)·관측 상태(actual_next_state) 필드는 **수치·구조 메타데이터만** 수집하며, 원문 텍스트는 수집 대상에서 제외됩니다. 해당 필드는 4KB 크기 제한이 적용됩니다.

회사는 법령에 따른 개인정보 보유·이용 기간 또는 이용자로부터 개인정보를 수집 시에 동의받은 개인정보 보유·이용 기간 내에서 개인정보를 처리·보유합니다.

- 회원 탈퇴 또는 보관 기간 만료 시 **30일 이내** 파기합니다.
- 단, 법령에 따른 보관 의무가 있는 데이터(AuditEvent 등)는 해당 기간 동안 별도 보관 후 파기합니다.
- 이미 집계·익명화된 벤치마크 통계값은 특정 개인을 식별하는 것이 기술적으로 불가능하므로 개인정보보호법 제2조 제1호의 개인정보에 해당하지 않아 삭제 요청 대상에서 제외됩니다.

## 4. 개인정보의 처리 위탁

### 4-1. 위탁 현황

회사는 서비스 제공을 위해 다음과 같이 개인정보 처리 업무를 외부에 위탁하고 있습니다.

| 수탁사 | 위탁 업무 | 처리 데이터 | 보유 및 이용 기간 |
|--------|---------|------------|:---------------:|
| **Amazon Web Services, Inc. (AWS)** | 클라우드 인프라, 데이터베이스(RDS), 스토리지(S3) 운영 | 이메일(암호화), 에이전트 진단 수치 데이터, 감사 로그 | 서비스 이용 기간 종료 후 파기 |
| **[결제 대행사]** *(추후 기재)* | 구독 결제 처리, 청구서 발행 | 이메일 주소 (결제 알림 목적) | 전자상거래법에 따라 5년 |

| **Anthropic, PBC** | AI 언어 모델(Claude API) — 진단 결과 해석, 개선 권고 생성 | 에이전트 진단 수치 데이터 (개인 식별 정보 제외) | API 처리 완료 즉시 (학습에 미사용) |

> **수탁사 추가 안내**: 결제 대행사 및 이메일 발송 서비스는 서비스 시행(2026년 7월 1일) 전 확정 후 기재합니다.

### 4-2. 위탁 관리

회사는 위탁 계약 시 「개인정보보호법」 제26조에 따라 다음 사항을 계약서에 명시하고 수탁사의 개인정보 보호 수준을 관리·감독합니다.

- 위탁 업무 목적 및 범위
- 재위탁 제한 조항
- 기술적·관리적 보호조치 의무
- 위탁 종료 시 개인정보 반환 또는 파기 의무
- 손해배상 조항
- 보안 사고 발생 시 24시간 내 통지 의무

### 4-3. AI 모델 활용 고지

회사는 AI 에이전트 진단 결과 해석 및 개선 권고 사항 생성에 Anthropic의 Claude AI 모델을 활용합니다.

- **처리 데이터**: 에이전트 진단 수치 데이터 (HAS 점수, 15개 지표값)
- **미처리 데이터**: 이메일, 회사명 등 개인 식별 정보는 AI 모델에 전달하지 않습니다.
- **학습 비사용**: Anthropic API를 통해 처리된 데이터는 모델 학습에 사용되지 않습니다. (Anthropic API 데이터 처리 방침 준수)
- **참고 자료**: AI 모델이 생성하는 진단 해석 및 EU AI Act·ISO/IEC 42001 준수 점수는 참고 정보이며, 공식 법적 인증을 구성하지 않습니다.

## 5. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다.

1. **이용자가 사전에 동의한 경우** (예: 공개 케이스스터디 동의)
2. **법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우**

> 법적 근거 없는 수사기관 요청은 즉시 거부하며, 법적 근거 있는 요청에 의해 제공한 경우 72시간 이내에 해당 이용자에게 통지합니다. 단, 법원의 통지 금지 명령이 있는 경우 명령 해제 즉시 통지합니다.

**완전 익명화된 집계 통계 데이터**(Level 2~3 벤치마크)는 개인정보에 해당하지 않으므로 제3자 제공 규정의 적용 대상이 아닙니다.

## 6. 개인정보의 파기 절차 및 방법

### 6-1. 파기 절차

회사는 개인정보 보유 기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체없이 해당 개인정보를 파기합니다.

```
파기 프로세스:

Step 1. 파기 대상 확인
  · 보관 기간 만료 데이터 목록 자동 추출 (매월 1일)
  · 이용자 삭제 요청 접수 (DELETE /v1/data/me 또는 이메일)

Step 2. 삭제 실행 (30일 이내 완료)
  Day 1~3:   신원 확인 (이메일 인증 또는 API 키 확인)
  Day 4~10:  Cold Store (S3) 원시 에피소드 로그 삭제
  Day 11~20: Hot/Warm Store (PostgreSQL) 데이터 삭제
             - 에피소드 기록, 진단 결과, 시계열 데이터, 행동 데이터 삭제
  Day 21~25: 계정 소프트 삭제 (이메일 원본 덮어쓰기, 회사명 null 처리)
  Day 26~30: 잔여 레코드 0건 검증 + 삭제 완료 통보

Step 3. 삭제 미대상 데이터 안내
  · AuditEvent(감사 로그): 법적 보관 의무로 삭제 불가 (3년 보관 후 파기)

```

## 8. 자동화된 결정에 관한 사항

### 8-1. 자동화된 결정 개요

회사는 다음의 자동화된 처리를 통해 이용자의 에이전트에 대한 진단 결과를 생성합니다.

| 자동화 처리 | 내용 | 법적 근거 |
|------------|------|---------|
| HAS 점수 산출 | 수집된 에피소드 데이터로 15개 지표를 계산하고 가중 합산하여 0~100점 HAS 점수 생성 | 계약 이행 |
| 등급 판정 | HAS 점수 기준 A+·A·B·C·D 등급 자동 부여 | 계약 이행 |
| 배포 적합성 판단 | 등급 기반 "전면 배포 가능" 등 권고 문구 자동 생성 | 계약 이행 |
| 개선 액션 추천 | AI 모델(Claude)을 활용하여 맞춤형 개선 권고 사항 자동 생성 | 계약 이행 |

### 8-2. 자동화된 결정의 의미와 한계

자동화 진단 결과는 통계적 계산과 AI 모델의 분석을 기반으로 합니다. 회사는 다음 사항을 명확히 안내합니다.

- **참고 정보**: HAS 점수와 등급은 에이전트의 World Model 품질을 측정하는 참고 지표입니다.
- **법적 판단 미구성**: EU AI Act·ISO/IEC 42001 준수 점수는 공식 인증 또는 법적 준수 판단을 구성하지 않습니다.
- **신뢰구간 제공**: 모든 HAS 점수에는 95% 신뢰구간(하한·상한)이 함께 제공됩니다.

### 8-3. 자동화된 결정 거부권

귀하는 서비스 설정에서 자동화된 진단 결과에만 의존하지 않도록 선택할 수 있으며, 전문가 직접 검토를 요청할 수 있습니다. 거부 방법: 서비스 설정 페이지 → 개인정보 관리 → 자동화 결정 설정

> **안내**: 위 수치 데이터는 에이전트의 기계적 행동 결과이며, 에이전트가 처리하는 실제 업무 콘텐츠(텍스트, 이미지, 개인정보 등)는 포함되지 않습니다.
