## 1. 인증 프로그램 개요

### 1.1 프로그램 목적

HAchillesWorld 인증 프로그램은 AI 에이전트를 **체계적으로 진단·최적화·운영**할 수 있는 역량을 객관적으로 검증하고 공인하는 제도다.

단순한 도구 사용법을 넘어, **Levels × Laws 프레임워크 이해 → 실제 에이전트 개선 → 자립적 운영** 역량을 3단계로 검증한다.

> **"측정할 수 없는 것은 개선할 수 없다. 인증받지 못한 역량은 신뢰받기 어렵다."**

### 1.2 인증의 가치

```
개인에게:
  • AI 에이전트 품질을 객관적으로 증명하는 자격증
  • 취업·프리랜서·컨설팅 경쟁력 강화
  • 커리어의 차별화 포인트

팀/조직에게:
  • 에이전트 개발·운영 역량의 공식 기준
  • 채용 및 역량 평가 기준
  • 고객에게 기술력을 증명하는 레퍼런스

에이전트 시스템에게:
  • L3 Evolver 인증 = 운영 신뢰성 공인
  • 투자자·파트너에게 기술 수준 증명
  • 규제 대응 시 기술 성숙도 근거
```

### 1.3 이론적 기반

HAchillesWorld 인증은 《Agentic World Modeling 2027》(박성훈 저)의 세 가지 이론을 기반으로 한다.

| 이론 | 인증에서의 역할 |
|------|--------------|
| **Levels × Laws 프레임워크** | 에이전트 수준 판정의 객관적 기준 |
| **Planning Depth (MCTS)** | 에이전트 지능의 깊이 측정 기준 |
| **Harness Engineering** | 안전한 에이전트 운영 역량 기준 |

#### Module 3: MCTS 계획기 통합 (Week 3–5) ⭐

| 차시 | 주제 | 핵심 코드 |
|------|------|---------|
| M3-1 | MCTS 이론 | Selection·Expansion·Rollout·Backprop |
| M3-2 | UCB1 탐색 정책 | `exploration_c` 최적화 |
| M3-3 | MCTSPlanner 구현 | `MCTSPlanner(max_depth=5)` |
| M3-4 | Planning Depth 확장 | Depth 3 → 15 단계적 업그레이드 |
| M3-5 | Latency 최적화 | 비동기 계획·캐싱 전략 |
| M3-6 | World Model 래퍼 | 도메인별 State 정의 |
| **실습** | Depth 15 달성 검증 | Latency ≤ 200ms 조건 |

```python
# HCE 실습 핵심 코드 예시
from hachillesworld.optimize.mcts import MCTSPlanner

planner = MCTSPlanner(
    agent_name="cert-agent",
    n_simulations=50,
    exploration_c=1.414,
    max_depth=15,           # HCE 목표: 15스텝
    rollout_policy="random",
    discount_factor=0.95,
)

# 검증: Depth 15 달성 + latency 측정
import time
start = time.perf_counter()
action = planner.plan(state, goal, world_model)
latency_ms = (time.perf_counter() - start) * 1000
assert latency_ms < 200, f"Latency {latency_ms:.1f}ms 초과"
```

## 부록 B: 자주 묻는 질문 (FAQ)

**Q1. HCP 없이 HCE에 직접 도전할 수 있나요?**  
→ 원칙적으로 순서를 따르도록 권고합니다. 단, 충분한 실무 경험(2년 이상)이 있는 경우 HCP를 면제 신청할 수 있습니다. 면제 심사 비용: ₩50,000.

**Q2. 시험은 한국어와 영어 중 어느 것으로 응시하나요?**  
→ 현재 한국어로만 제공됩니다. 2026년 4분기 영어 버전 출시 예정.

**Q3. 실습 과제에서 실제 업무 데이터를 사용해도 되나요?**  
→ 가능합니다. 단, 제출 전 개인정보·영업비밀을 반드시 익명화해야 합니다. 제공 샘플 데이터 사용도 가능합니다.

**Q4. HCAN 갱신을 놓치면 어떻게 되나요?**  
→ 자동 만료됩니다. 만료 후 90일 이내에는 갱신 절차로 복원 가능합니다. 90일 초과 시 HCE부터 재취득해야 합니다.

**Q5. 에이전트 시스템의 L3 인증과 개인 HCAN 인증은 다른가요?**  
→ 다릅니다. 개인 HCAN은 사람의 역량 인증이고, 에이전트 L3 인증은 시스템의 품질 인증입니다. HCAN 보유자가 L3 인증 심사에 참여할 수 있습니다.
