## 데이터 수집 동의서

### 연구 참여 동의

저는 HAW-STUDY-001 연구 참여에 자발적으로 동의하며, 다음 사항을 확인합니다.

#### 수집 데이터

| 항목 | 내용 | 용도 |
|------|------|------|
| 에피소드 성공/실패 여부 | bool | HAS 지표 산출 |
| 에이전트 확신도(confidence) | 0~1 | SCR, ECE 계산 |
| 계획 깊이(planning_depth) | 정수 | PD 지표 계산 |
| 도구 호출 목록 | 문자열 리스트 | AS 지표 계산 |
| 실행 시간(duration_ms) | 밀리초 | IRT, HC 계산 |
| KPI 지표 | 제출한 수치 | H1 가설 검정 |
| 에이전트 ID | SHA256 해시 | 에피소드 연결 |

#### 수집하지 않는 데이터

- 에이전트 프롬프트 원문
- 고객 개인정보
- 영업 기밀, 계약 정보
- 직원 성명, 이메일, 연락처
- 시스템 접속 로그 (인증 정보 등)
